Security

This portal runs entirely in the browser (static hosting). The password hash lives in admin/config.js (gitignored). Copy config.example.js, generate a new hash, and never commit real credentials.

Access URL: /admin/ — not linked from the public portfolio.

Generate password hash:
node -e "console.log(require('crypto').createHash('sha256').update('YOUR_PASSWORD').digest('hex'))"
